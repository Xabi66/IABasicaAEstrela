# Descrición

Escena que genera un laberinto al iniciar el juego. Cuando se pulsa la tecla P crea un nodo de inicio y uno de fin, y cada vez que se pulsa C va actualizando las rutas disponibles desde el nodo inicial hasta hallar una que llegue al nodo final.

# Título principal

Resolución de un laberinto.

## Subtítulo

Lista de cambios realizados:

-Se redujo a la mitad ell ancho, alto y escala del laberinto.

-Se ajusto la camara para que fuera visible todo el laberinto y la información de cada nodo.
